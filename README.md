# DIKWP AccordMesh OS

**Active Inter-AI Federation for Semantic Exchange, Energy-Aware Coordination, Purpose Covenants and Auditable Alliances.**

AccordMesh connects the DIKWP semantic mesh, purpose assurance, multi-agent coordination, reciprocal communication and public-governance work into a bounded inter-AI federation runtime.

## Quick start

```bash
python -m unittest discover -s tests -v
python src/accordmesh_demo.py --scenario research
python src/accordmesh_local_gateway.py
```

Open `prototype/index.html` for a no-server browser demonstration.

## Non-goals

The reference runtime does not call external models, hold API keys, self-replicate, acquire resources, execute high-impact actions, form secret alliances or claim conformance with A2A/MCP.
